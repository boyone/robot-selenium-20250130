## [robocop](https://robocop.readthedocs.io/en/stable/)

1. Install `robocop`

   ```sh
   pip install robotframework-robocop
   ```

2. Usage

   ```sh
   robocop
   ```

   ```sh
   robocop -r all
   ```

   ```sh 
   robocop --output robocop.log .
   ``` 

   ```sh
   robocop --list
   ```

---

## [robotidy](https://robotidy.readthedocs.io/en/stable/)

1. Install `robotidy`

   ```sh
   pip install robotframework-tidy
   ```

2. Usage

   ```sh
   robotidy .
   ```

   ```sh
   robotidy --diff --check .
   ```

   ```sh
   robotidy --list
   ```
