# Install Robotframework and SeleniumLibrary

1. Install Robotframework

   ```sh
   pip install robotframework
   ```

2. Install Robotframework-SeleniumLibrary

   ```sh
   pip install robotframework-seleniumLibrary
   ```

3. Check Installation

   ```sh
   pip list
   ```
